Compiled TSPROC Executables and Source Code


Executables

TSPROC executables are currently available for Windows (32-bit) systems and Linux (64-bit) systems. There is no installation procedure needed; unzip or untar the archive files and ensure that the executables are placed into your path and are accessible from the command line.

Version 1.0

  • Windows (32-bit): http://wi.water.usgs.gov/models/tsproc/downloads/tsproc_windows_x86.zip, 481KB, compiled October 26, 2012
  • Windows (64-bit): http://wi.water.usgs.gov/models/tsproc/downloads/tsproc_windows_x64.zip, 565KB, compiled October 26, 2012


Source Code

A ZIP file that contains all of the source code for TSPROC may be downloaded here: http://wi.water.usgs.gov/models/tsproc/downloads/tsproc_source_git.zip 17.8MB



Source Code Documentation

Documentation for the current version of the source code may be viewed here: http://wi.water.usgs.gov/models/tsproc/tsproc_code_docs/
